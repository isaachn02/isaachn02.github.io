Ejercicio 1

Ejercicio 7

Define una función que reciba un número y devuelva su factorial.

Ver solución
def factorial(numero):
    resultado = 1

    for i in range(1, numero + 1):
        resultado = resultado * i

    return resultado


numero = int(input("Introduce un número: "))
print("El factorial es:", factorial(numero))

Ejercicio 8

Define una función que determine si un número es primo.

Ver solución
def es_primo(numero):
    if numero <= 1:
        return False

    for divisor in range(2, numero):
        if numero % divisor == 0:
            return False

    return True


numero = int(input("Introduce un número: "))

if es_primo(numero):
    print("El número es primo")
else:
    print("El número no es primo")

Ejercicio 12

Crea un diccionario con alumnos y notas, y calcula la media.

Ver solución
alumnos = {
    "Ana": 8,
    "Luis": 6,
    "Marta": 9,
    "Pedro": 7
}

suma = 0
contador = 0

for alumno in alumnos:
    suma = suma + alumnos[alumno]
    contador = contador + 1

media = suma / contador
print("La media de las notas es:", media)
