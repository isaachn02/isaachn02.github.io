Ejercicio 9

Define una función que reciba un número y devuelva su factorial.

Ver solución 
def factorial(numero):
resultado = 1

for i in range(1, numero + 1):
    resultado = resultado * i

return resultado

numero = int(input("Introduce un número: "))

print(
"El factorial es:",
factorial(numero)
)

Ejercicio 10

Define una función que determine si un número es primo.

Ver solución 
def es_primo(numero):
if numero <= 1:
    return False

for divisor in range(2, numero):
    if numero % divisor == 0:
        return False

return True

numero = int(input("Introduce un número: "))

if es_primo(numero):
print("El número es primo")
else:
print("El número no es primo")
